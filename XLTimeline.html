How to edit

  • Open in any text editor. Find the DATA section.
  • Change PROGRAM_START to your real start Monday.
  • Change TOTAL_WEEKS (e.g. 52 for a full year).
  • Edit TRACKS: change name and owner fields only.
  • Edit EVENTS: adjust startWeek / endWeek numbers.

Tips

  • Click an event label to expand its sub-tasks.
  • Hover any bar to see a details tooltip.
  • Set TODAY_WEEK to highlight the current week.
  • Colors: c1=amber, c2=teal, c3=violet, c4=red, c5=green.
